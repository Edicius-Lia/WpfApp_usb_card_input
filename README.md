# WpfApp_usb_card_input

## 项目简介

本项目是一个基于 WPF 的 USB 扫描枪输入捕获工具，支持全局键盘钩子，能够在应用无焦点时捕获扫描枪输入的条码数据。适用于需要自动采集条码的桌面应用场景。

## 主要功能

- **全局键盘钩子**：无需窗口焦点即可捕获键盘和扫描枪输入。
- **条码数据处理**：自动识别扫描枪输入，区分手动输入，提取有效条码信息。
- **事件通知**：通过事件机制将条码数据传递给外部业务逻辑。
- **兼容性强**：支持 .NET 8，解决钩子注册兼容性问题。
